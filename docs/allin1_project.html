R Package: allin1

R package built for ADC 405. Package website · GitHub repo

Overview

allin1 is an R package that bundles shortcuts for common but tedious data-wrangling tasks, the kind of small multi-step operations you end up rewriting constantly. Each function collapses a familiar dplyr pattern into a single call. The package currently exports three functions:

  • sort_mutate() — create a new column and sort the data frame by it in one step.
  • drop_na_rows() — remove every row containing a missing value.
  • top_n_percent() — keep the top N percent of rows by a chosen variable.

The package is fully documented (with a vignette and per-function reference) on its pkgdown website; this page is a quick tour.

Installation

# install.packages("devtools")
devtools::install_github("ADC-405-S26/allin1")
library(allin1)

Usage

The examples below use employees, a small demo dataset that ships with the package.

sort_mutate()

Add an earnings column (hours_worked * hourly_wage) and sort by it, highest first — a mutate and an arrange in a single call:

employees |>
  sort_mutate("earnings", hours_worked * hourly_wage, desc = TRUE)
     name hours_worked hourly_wage earnings
1 Charlie           45          50     2250
2   Alice           40          30     1200
3     Bob           35          25      875
4   David           20          20      400
5    <NA>           NA          NA       NA

drop_na_rows()

Drop any row with a missing value — handy for quick cleanup before an analysis:

employees |>
  drop_na_rows()
     name hours_worked hourly_wage
1   Alice           40          30
2     Bob           35          25
4 Charlie           45          50
5   David           20          20

top_n_percent()

Keep the highest-paid 50% of employees by hourly_wage:

employees |>
  top_n_percent(hourly_wage, percent = 50)
     name hours_worked hourly_wage
1 Charlie           45          50
2   Alice           40          30

## Overview

`allin1` is an R package that bundles shortcuts for common but tedious
data-wrangling tasks, the kind of small multi-step operations you end up
rewriting constantly. Each function collapses a familiar `dplyr` pattern into a
single call. The package currently exports three functions:

- **`sort_mutate()`** — create a new column and sort the data frame by it in one step.
- **`drop_na_rows()`** — remove every row containing a missing value.
- **`top_n_percent()`** — keep the top *N* percent of rows by a chosen variable.

The package is fully documented (with a vignette and per-function reference) on its
[pkgdown website](https://adc-405-s26.github.io/allin1/); this page is a quick tour.
